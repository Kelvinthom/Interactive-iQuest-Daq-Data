iQuest-Vib-Daq Data Specification

Data Acquisition structure and parameter definitions for cloud ingestion.

Overview

The iQuest-Vib-Daq system provides vibration and condition monitoring data for predictive analytics and fault detection.

DAQ_Info

Metadata describing the acquisition setup and sensor configuration.

Location: Physical site
assetID: Machine UID
Units: g, mm/s, °C, Hz

Example JSON Structure


                        

                    


                    

                        

                            
S_aRMS_t / S_vRMS_t

                            
Time-domain RMS signals representing overall vibration intensity.

                        

                        

                            
S_aTW / S_vTW

                            
Raw Time Waveform data used for transient waveform analysis.

                        

                        

                            
S_aFFT / S_vFFT

                            
Frequency-domain FFT used for identifies mechanical fault signatures.

                        

                        

                            
S_temp / speedRMS

                            
Temperature and rotational stability measurements.
