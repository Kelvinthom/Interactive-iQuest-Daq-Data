iQuest-Vib-Daq Data Specification

Overview

The iQuest-Vib-Daq system provides vibration and condition monitoring data for predictive analytics and fault detection. This document defines the structure, parameters, and data types available for cloud ingestion and analytics processing.

Data Streams

The Data Acquisition (DAQ) system generates the following primary data types.

DAQ_Info

Metadata describing the acquisition setup, monitored asset, and sensor configuration.

Fields

  • Location: Physical or operational site of the asset.
  • assetID: Unique identifier of the monitored machine.
  • sensorLabels: Human-readable names describing sensor positions.
  • sensorID: Internal identifiers mapped to each sensor channel.
  • Units: Measurement units (g, mm/s, degrees C, ms, Hz).
{
  "DAQ_Info": {
    "Location": "BM-Primary-Milling",
    "assetID": "Ball-mill-3",
    "sensorLabels": ["MTR-NDE-H", "G/OU-DE-V", "MTR-DE-H", "G/IN-DE-H", "G/IN-NDE-H", "G/OU-NDE-H", "PN-DE-H", "PN-NDE-H"],
    "sensorID": ["POINT1", "POINT2", "POINT3", "POINT4", "POINT5", "POINT6", "POINT7", "POINT8"],
    "Units": ["g", "mm/s", "degrees C", "ms", "Hz"]
  }
}

S_aRMS_t

Time-domain Root Mean Square (RMS) of acceleration signals. Represents overall vibration energy or intensity over time.

S_vRMS_t

Time-domain Root Mean Square (RMS) of velocity signals. Commonly used to assess machine vibration severity.

S_aTW

Acceleration Time Waveform data. Raw or processed acceleration signal sampled over time.

S_vTW

Velocity Time Waveform data. Time-based velocity signal used in condition monitoring.

S_aFFT

Frequency-domain representation (FFT) of acceleration signals. Used to identify specific vibration harmonics.

S_vFFT

Frequency-domain representation (FFT) of velocity signals. Commonly used in rotating machinery diagnostics.

S_temp

Temperature measurements captured by the DAQ system for thermal monitoring.

speedRMS

RMS value of rotational speed or speed variation. Indicates operational stability.
